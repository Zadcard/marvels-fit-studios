05

Building any screen

Layout rules
  • Shell: centered app card on #e3e0dd backdrop, #f1efec inner, r34. Fixed 1400px design, auto-scaled to fit viewport width.
  • Top bar: logo · centered nav pills · action cluster (AI, alerts, chat, divider, avatar).
  • Body: optional left icon rail + content. Content leads with breadcrumb → H1 + Live pill → right-aligned actions.
  • Cards on white, 18px gaps, 20–22px padding, 1px #edecea borders — no drop shadows on cards.
Do & don't
  • Do use Space Grotesk for every number & heading; red for the single primary action per view.
  • Do use emerald icon chips + green/amber/orange/red for status only.
  • Don't introduce new hues, cold grays, heavy shadows, or a second red-weight primary button per view.
  • Don't reference "floors/rooms" — this is a studio of group & private sessions, coaches, clients.
