Rules
Layering. Canvas #1B1B1C → panel #0C0C0D (r28) → surface #161618 (r20). Never borders between layers; contrast does the separation.
Red budget. One red CTA + one active nav item + live data marks per screen. Everything else stays ink and pastel.
Pastels never speak. Rose/mint carry data and highlights only — never body text, never buttons. Text on pastel is dark ink (#1D0809).
